The Opportunity
We're looking for a Junior to Level 2 Full Stack Developer to join our engineering team.

You'll work across the full stack building features, fixing issues, and contributing to a rapidly evolving platform. You'll also have the opportunity to take ownership of core components and leverage AI agents to accelerate development output.

This role suits someone with 1–2 years' experience who is curious, proactive, and motivated to grow fast.

You'll be supported by a senior developer, but you must be comfortable working autonomously and taking initiative.

What You'll Do
- Build and maintain features across the full stack (UI, API, database)
- Work on modern front-end interfaces using frameworks such as React and Bootstrap-based UI components
- Contribute to backend systems built with .NET (C#) and API-driven architectures
- Write and optimise queries against SQL databases (e.g. Postgres, SQL Server, MySQL)
- Use AI coding tools (Copilot, Claude, Cursor, etc.) as a core part of your workflow
- Participate in code reviews — giving and receiving practical feedback
- Debug and resolve issues across staging and production environments
- Contribute ideas to improve system performance, scalability, and developer efficiency
- Document key features and technical decisions

What We're Looking For
Experience: 1–2 years of hands-on full stack development (commercial, freelance, or strong project work)
Comfortable working across both front end and back end

Technical Skills

Back End: Exposure to .NET / C# or similar backend frameworks, building and consuming REST APIs
Databases: Working knowledge of SQL (queries, joins, basic schema design)
General: Git (branching, pull requests, code reviews), cloud platforms (Azure preferred)

AI Tooling (Important): Actively use AI coding assistants (Copilot, Claude Code, Cursor, etc.)
