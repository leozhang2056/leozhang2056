Intermediate Software Engineer
Vista Group - Horizon Platform
Auckland, New Zealand
Full time

This Intermediate Software Engineer role sits within one of our Enterprise Stream engineering squads, contributing to Horizon, Vista's cloud-hosted data warehouse product.

You'll be part of a skilled, collaborative team, working primarily on backend services, data pipelines, and SQL-backed APIs, helping ensure Horizon remains scalable, reliable, and performant for customers worldwide.

The role:
- Use your C# and SQL expertise to design, build, and maintain backend services and APIs that support Horizon's data warehouse platform
- Develop and contribute to platform capabilities with a strong focus on performance, reliability, security, and maintainability
- Collaborate closely with your squad, product managers, and architecture partners to deliver high-quality solutions
- Contribute to code reviews, testing, and continuous improvement practices

Requirements:
- C# and ASP.NET Core (.NET 6+), with .NET and C# as core skills
- SQL Server, including query optimisation and performance tuning
- Building and consuming API integrations
- Working with data-intensive applications or data warehouse-style systems
- Microsoft Azure (application hosting, data services, or monitoring)
- Working understanding of CI/CD pipelines and containerised environments (e.g. Docker, Kubernetes, Octopus)
- Strong problem-solving and debugging skills
- Collaborative mindset and clear communication style
- Interest in improving platform stability, scalability, and performance

About Vista: software for the cinema industry, HQ in Auckland, offices worldwide.
