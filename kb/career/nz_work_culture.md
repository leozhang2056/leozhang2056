# 新西兰职场文化

> 来源：AUT Employability & Careers

---

## 企业特征

- 平均 **13 人**，扁平结构
- 全员多面手，独立工作
- Can-do attitude 受重视

## 管理风格

- 不正式，直呼其名
- Manager 是 team member
- 鼓励贡献 ideas 和 feedback

## 职场社交

- Morning tea、生日、After work drinks
- 建立 bond 和 network 的组成部分

---

## 雇主看重的软技能

Communication / Teamwork / Curiosity / Collaboration / Critical Thinking / Problem Solving / Customer Focus / Adaptability

---

## 为什么 Network 重要

- 公司小 → 雇人靠信任
- 隐藏市场 70%
- 人脉来源：AUT Events / 爱好 / 同学老师 / Toastmasters / Meetup / LinkedIn / Clubs

---

## 本地经验

NZ 雇主重视本地经验因为：NZ referee + 懂职场文化 + 专业技能

**志愿者经验 = 本地经验**

## 求职路径

- 被动：投广告岗位 + NZ 格式 CV/CL + LinkedIn
- 主动：Personal network / Professional network / 直接联系公司

## 国际学生

- 清楚签证工作权限
- 了解最低工资和雇员权利
- NZ CV 格式 ≠ 原国家格式
- 先做志愿者积累本地 referee
